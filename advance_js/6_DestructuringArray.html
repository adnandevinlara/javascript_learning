Destructuring arrays

This feature comes in ES6 version on javascript

If we wan to assign arrays value to variables than for that we need to write lot of code,if there are many data in array. forexample

let user = ['adnan zaib',25];
let name = user[0];
let age = user[1];
// and so on

In ES6 version new feature introduced of destructuring and minize the syntext

let[name,age] = user;
